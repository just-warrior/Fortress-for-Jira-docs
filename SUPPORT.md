# Support

**Last Updated:** June 10, 2026

For support, bug reports, installation questions, Marketplace review questions, privacy questions, or security issues related to **Jira Fortress for Jira**, contact:

**JMC Labs**  
Email: **support@jmclabs.dev**

## What to Include

When requesting support, include:

- Jira Cloud site URL
- Jira project key where the issue occurred
- whether the project is included in the Jira Fortress allowlist
- app version if available
- browser and operating system
- a brief description of the issue
- screenshots or screen recordings if relevant

Do not send passwords, API tokens, Personal Access Tokens, or other secrets.
