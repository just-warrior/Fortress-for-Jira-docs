# Jira Fortress for Jira

Welcome to the documentation for **Jira Fortress for Jira**, a Forge-native Jira Cloud app by **JMC Labs**.

Jira Fortress adds a lightweight turn-based fortress battle to Jira issues. Jira administrators can enable the app only for selected projects, and users can open a battle from an issue panel. The battle result can update the issue assignee according to the game outcome.
